## Feedr

Hi, this is a simple food app created with Next.js

If you would like to start the app first, run the development server:

```bash
pnpm dev
```

And then open [http://localhost:3000](http://localhost:3000)

## Stack

This project is using:

- Next.js
- TailwindCSS
- Mongo + Mongoose
- NextAuth.js
- React Hook Form + Yup
- SWR
- Amazon S3
